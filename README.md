# What is this ?

Get perfect linear gradient colors for non-designer .

# Installation 

`npm i navilalith --save`

Then...

```
import { navilalith } from 'navilalith';

navilalith({
    color_type: 'greenpink'
});



```

## Options

* *color_type* - greenpink
* *color_type* - greenaqua
* *color_type* - greenyellow
* *color_type* - greenorange
